# エグゼクティブサマリ  
ピアノ、ギター/ベース、ドラムはそれぞれ固有の物理モデル式で音響的特性が再現できる。**ピアノ**は張力付き剛性弦（Euler–Bernoulli梁）のPDEとハンマー衝突、倍音ずれ（インハーモニシティ）と多弦ビート、2段減衰を含むモーダル合成でモデル化される。**ギター/ベース**は理想弦または剛性弦の波動方程式で弦振動を表し、ピックや指での撥弦励振＋ボディ共鳴で音色を作る。弦長・張力・密度から倍音周波数が決まり、打点による励振スペクトル（例：位置依存の$\sin(n\pi x/L)$）と指・ピックノイズが加わる。**ドラムセット**は2次元膜の波動方程式（固定端円形膜）で基礎振動モードを求め、スティック衝撃による初期変位（あるいは速度）で打撃。膜振動はBessel関数系で表現され、スネア金具や殻の共鳴はIR/FDNで処理する。各楽器とも、シンプル実装式（モーダル合成やウェーブガイドモデル）と、高精度物理モデル（PDE＋ハンマー/ピックモデルなど）の両面から設計可能である。

## ピアノ  
- **最小実装式（表）**  

| 要素       | 数式例                            | 説明                              |
|------------|-----------------------------------|-----------------------------------|
| 振幅・倍音  | $a_{n,s}(v)$                     | ベロシティ依存の各倍音振幅 |
| 周波数      | $f_{n,s}=n f_1 \sqrt{1+B_s n^2}$ | 第1部分音$f_1$とインハーモニシティ$B_s$ |
| 減衰       | $e^{-t/\tau_{n}}$ or 2段減衰    | 簡易的には指数減衰、実際は「速/遅」2段階減衰でビート再現 |
| ビート      | $\sum_{s=1}^{N_s}\sin(2\pi f_n(1+\delta_s)t)$ | ほぼ同調した複数弦の重ね合わせでビートが生じる |
| ハンマー   | $F_h(t)=K_h[x_h(t)-\eta(x_\mathrm{h},t)]_+^\alpha$ | ハンマー力学モデル（非線形バネ） |
| 響板共鳴   | $h_{\rm board}(t)*(\cdot)$        | 響板・筐体のインパルス応答で畳み込み |
| 雑音項     | $T(t,v)+N(t,v)$                  | 打撃音やフェルト・空気ノイズ等の付加成分 |

- **精密モデル（式と説明）**  
  張力$T$を受ける鋼弦は弦の剛性も考慮したEuler–Bernoulli梁モデルでPDEを満たす：  
  $$\rho A\,\frac{\partial^2\eta}{\partial t^2}=T\,\frac{\partial^2\eta}{\partial x^2}-EI\,\frac{\partial^4\eta}{\partial x^4}-\dots$$  
  ここで$\rho A$は質量（線密度×断面積）、$EI$は曲げ剛性（$E$:ヤング率、$I$：断面2次モーメント）である。剛性項により高次の波速が速くなり、倍音周波数が整数倍からずれる（インハーモニシティ）。剛性小さい場合、固有振動数は次の近似式で与えられる：  
  $$
  f_n = n f_0\sqrt{1 + B n^2},\qquad
  B = \frac{\pi^2 EI}{T L^2}\approx\frac{\pi^3 E d^4}{64T L^2},
  $$  
  ただし$L$は弦長、$d$直径（全断面の曲げ慣性半径）、$B$はインハーモニシティ係数。理想弦なら$B=0$で$nf_0$の等比倍音列となる（例: $f_0=n/(2L)\sqrt{T/\rho}$）。実装では、基準音$F$と$B$値に応じて高域倍音周波数を補正することでインハーモニシティを表現できる。  

  実際のピアノ音は複数弦・複数偏極モードの重畳で**ビート**と**二段減衰**を示す。２段減衰とは、音量の早い減衰とゆるやかな減衰が連続して現れる特性で、複数弦の結合や縦横振動の異なる減衰に起因する（Weinreich 1977など）。  
- **励振・ノイズ項**  
  ハンマー衝突により弦が励振される。衝突時の接触位置$x_{\rm strike}$、速度$v$、ハンマー特性が音色に影響する。簡易モデルでは、衝撃を一時的な力$F_h(t)$や初期変位として入力する。ハンマーの非線形バネモデル（Hertz型）では$F_h=K_h[x_h(t)-\eta(x_h,t)]_+^\alpha$の形で表現される。速い打鍵では高域の励振強度が上がる（励振スペクトル幅の拡大）ため、ベロシティ依存の振幅・スペクトルシェーピングが必要となる。打鍵時の機構音（鍵盤・アクション音）や打弦位置に近い共鳴・付加音もトランジェントとして付加する。  
  **ノイズ項**としては、弦とハンマー・フェルト間の摩擦音やフェルトのファズ音、弦/響板間の擦れ音、ペダルやサステイン共鳴音などが含まれる。これらは独立のノイズ源として加算・付加濾波する。  

- **共鳴/ボディ処理**  
  ピアノの響板・金属フレーム・筐体は音響共鳴器として機能する。響板の振動は高密度かつ複雑なため、実測IRやモーダル合成、FDN（フィードバックディレイネットワーク）で近似するのが実装上現実的。Bank論文では響板を形状フィルタ付きFDNとし、ハンマー音と響板応答を分離してモデル化している。簡易には、響板のインパルス応答$h_{\rm board}(t)$をMIDIノートごとに畳み込み、出力に響き効果を付加する手法がとられる。  

- **パラメータ推定法**  
  弦の物理パラメータ（$L,T,\rho,E,d$など）は、製造仕様や調弦値から求められる。張力$T$は開放弦の基本周波数$f$と長さ$L$, 線密度$\mu=\rho S$から$T=(2Lf)^2\mu$で計算できる。インハーモニシティ係数$B$は弦材・径$d$・長さから上式で概算されるか、録音スペクトルから高次倍音の偏差量$f_n^2/(n^2f_0^2)-1$と$n^2$の回帰で推定する方法もある。ヤング率$E$や張力$T$は文献値や測定による。  

- **実装メモ**  
  基本モデルはモーダル合成（ダンピング正弦波列）やウェーブガイドで実装し、リアルタイム処理する。モード数$N$を揃え、各モードを減衰振動子$q_n(t)$として解けば、$$y(t)=\sum_{n,s}a_{n,s}e^{-t/\tau_n}\sin(2\pi f_{n,s}t+\phi_{n,s})$$という形で音源を生成できる。低音部は弦数が多く計算負荷が高いため、モード縮約や混合アプローチ（主要モードは精密、残余は簡易モデル）で計算量を減らす。ハンマー衝突はインパルス応答畳み込みまたは短時間FFTで加える手法が多く、共鳴は実測IR/FDNで簡易化する。  

## ギター・ベース  
- **最小実装式（表）**  

| 要素    | 数式例                                       | 説明                               |
|---------|----------------------------------------------|------------------------------------|
| 振幅・倍音 | $a_n(v)\sim\sin(n\pi x_{\rm strike}/L)$      | 打弦位置$x_{\rm strike}$依存で高音強度を変化させる（励振スペクトル） |
| 周波数   | $f_n=n f_0\sqrt{1+B n^2}$ または $nf_0$      | 基本周波数$f_0$とインハーモニシティ（ギター弦もわずかに剛性あり） |
| 減衰    | $e^{-t/\tau_n}$                             | 角周波数依存の減衰（通常は周波数高いほど減衰早い）   |
| ピック   | $\delta$関数的衝撃 または エンベロープ付FFT | ピック/指弦モデルによるトランジェント   |
| 共鳴    | ボディIR (モーダル合成/FDN)                  | ギター胴共鳴によるフィルタリング      |
| ノイズ   | 指弦ノイズ、フレットノイズ、ピックノイズ等   | スクレッチ音やシズル音   |

- **精密モデル（式と説明）**  
  ギター・ベースの弦も張力付き弦であり、基本的には波動方程式を満たす理想弦モデルが基盤となる。ヤング率や弦径を考慮した剛性弦モデルも同様に適用でき、上記ピアノ弦と同じ式が使える。弦の基本振動数は$$f_0=\frac1{2L}\sqrt{\frac{T}{\mu}}\ ,$$ ここで$\mu=\rho S$（線密度）。効果的弦長$L$はフレット位置・カポなどにより変化し、張力$T$も撥弦時の大振幅で変化する（非線形張力効果）。  

  モード合成的には、各モード$n$の振動子$q_n(t)$として以下で近似できる：  
  $$\ddot q_n + 2r_n \dot q_n + \omega_n^2 q_n = F_{\rm excite}(t)\sin\!\Bigl(\frac{n\pi x_{\rm strike}}{L}\Bigr),$$  
  ここで$\omega_n=2\pi f_n$、$r_n$は減衰定数、$F_{\rm excite}(t)$は撥弦力に相当する時間関数である。この式をフーリエ合成すれば、**Karplus–Strong**型ウェーブガイドやFDNでも類似効果が得られる。  

- **励振・ノイズ項**  
  ピックや指で弦を撥く動作は、一時的な力$F(t)$や初速分布として弦に与えられる。撥弦位置$x_{\rm strike}$が励振スペクトルを決定し、打撃の強さ（速度）で全体の音量と高域強度が変化する。例えば、打点スペクトルは$\sin(n\pi x_{\rm strike}/L)$で高次モードの励起度を調整する。ピックの硬さ・形状や撥弦角度も励振特性に影響し、これらは経験的にゲイン関数$G(v)$や切断周波数$f_c(v)$のパラメータで調整できる（強く弾くほど高域が増える）。ノイズとしては、弦をフレットに当てるピック音、指の擦れ音、弦鳴りのスナップ音、ホワイトノイズ様のアタックノイズなどがある。これらは音源信号に加算して金属音を付加する。  

- **共鳴/ボディ処理**  
  ギター/ベースでは胴体（サウンドホールを含む胴共鳴体）が音響的共鳴器となる。一般には実測共鳴周波数でモーダル合成を行うか、室内で録音したインパルス応答で畳み込む。アクティブ電子ギター・ベースの場合はピックアップ共振（キャパシタ・インダクタモデル）も考慮する。木製共鳴胴は複雑だが、低域は板の剛体振動、高域は内部波動と簡易FDMまたはFDNで近似できる。モーダル合成では胴体モードを$f_m,\tau_m$の組でモデル化し、弦振動にフィルタ的に作用させる。  

- **パラメータ推定法**  
  弦長$L$はフレット位置・スケール長、線密度$\mu$は弦ゲージから計算できる。張力$T$は開放弦の周波数と$L,\mu$から$T=(2Lf_0)^2\mu$で求まる。インハーモニシティ$B$も同様に計算可能で、巻弦では芯線・外径から精密計算する方法がある。単弦型の場合は$B$無視で良いが、トレモロやフレットずれによる微小音程差をモデル化するなら$B$や$\delta_s$（複数弦チューニング差）を導入する。パラメータ同定には弦振動のFFTスペクトルを使った回帰や、同定アルゴリズム（遺伝的計算など）が研究されている。  

- **実装メモ**  
  代表的にはカープラス・ストロング法（デジタルウェーブガイド＋ローパスフィルタ）でギター音源が実装されている。モーダル合成なら、主要数本のモード振動子を並列実装する。ベースは低音域なのでモード数少なく、リアルタイム実装しやすい。ノイズやアタック音は別途加える。エレキギターでは歪み・アンプ特性も加味する必要があるが、物理モデルのみなら不要。  

## ドラムセット  
*図: 円形膜の基準振動モード(0,1)*。ドラムヘッド（膜）は2次元波動方程式でモデリングできる。半径$a$の円形膜では、速度定数$c=\sqrt{T/\sigma}$（$T$:膜張力、$\sigma$:面密度）を用い、境界$u(a,\theta,t)=0$で以下のPDEを満たす：  
$$\frac{\partial^2u}{\partial t^2} = c^2\Bigl(\frac{\partial^2u}{\partial r^2}+\frac1r\frac{\partial u}{\partial r} +\frac1{r^2}\frac{\partial^2u}{\partial\theta^2}\Bigr)\ ,$$  
ここで$(r,\theta)$は極座標、$u(r,\theta,t)$が膜の変位である。このPDEは基底・原点付近をノードとする**ベッセル関数**系の固有モード解を持ち、（0,1）モード（外周ノードのみ）から高次の径方向・角方向節をもつモードが無数に存在する。モード固有周波数は、$J_m(\alpha_{mn} r/a)$の零点$\alpha_{mn}$で決まる（(0,1)モードは$\alpha_{01}\approx2.405$など）。各モード固有振動数は$k_{mn}=\alpha_{mn}/a$ に対し$f_{mn}=c\,k_{mn}/(2\pi)$で与えられる。  

- **最小実装式（表）**  

| 要素      | 数式例                          | 説明                             |
|-----------|---------------------------------|----------------------------------|
| 振幅・減衰  | $A_{mn}\exp(-t/\tau_{mn})$      | 模様$(m,n)$モードの初期振幅と減衰 |
| 周波数     | $(\alpha_{mn}/2\pi a)\sqrt{T/\sigma}$ | 根号形の固有周波数（ベッセル零点$\alpha_{mn}$） |
| 打撃       | インパルス勢い/ウェーブレット      | スティック衝撃による瞬間的励起    |
| スネア     | $*$フィルタ（ワイヤ・共鳴体）     | スネア用バネ振動や共鳴体との結合   |
| 共鳴殻     | 共鳴共振$h_{\rm shell}(t)$       | ドラム殻の共鳴応答（IR畳み込み）   |
| ノイズ     | ホワイトノイズ＋短調            | リム・ショット音、空気ノイズ      |

- **精密モデル（式と説明）**  
  円形膜では、弦の1D波動方程式の2D版を解く。固定端円板の共振振動は、ラジアル番号$n$と角方向次数$m$によるモード$(m,n)$で表され、任意初期条件はこれらのモード重畳で記述できる。膜振動のPDEにより、各モードの周波数$f_{mn}$と振動形状が決まる。実装上は有限要素やモーダル和で近似する場合、**簡略モード合成**を使い、基礎的な数個のベッセルモードを加算するのが現実的である。膜の質量$\sigma$、張力$T$は材質とテンションから決まる。具体例として中心打撃では(0,1),(1,1),(2,1)...が励起されやすい（図）。  

- **励振・ノイズ項**  
  スネアドラム等ではスティックやブラシで打撃し、弾み型ハンマー衝突と似たインパルス応力が膜に与えられる。初期条件として$u(r,\theta,0)$や$\partial_tu(r,\theta,0)$で表現可能で、例えばスティック先端領域で短い時間に押し込むモデルなどが用いられる。付加的に、ドラムはリムショット（縁打ち）音やブラシノイズ、空気移動音がある。スネアドラムでは胴下部のスネアワイヤが振動し、高域にパルス的「スナップ」音を加える。これらは独立にサンプリング・合成した音をミックスするか、フィルタ処理で模する。  

- **共鳴/ボディ処理**  
  ドラムヘッドの振動は背面共鳴ヘッドや金属枠（フープ）で反響・フィードバックを受ける。共鳴胴の振動は別個の共振器としてモデル化できる。例えば簡易的に一つのFRNでリバーブを付加したり、多数のテンション付ヘッドモードとFDNを組み合わせる手法がある。実用では全打撃IRライブラリによる畳み込みや、モード・マッチング法が使われる。金属のスネアワイヤやシンバル等は別モデル（弦や円板）で扱う。  

- **パラメータ推定法**  
  膜張力$T$や質量密度$\sigma$は製造値から概算できる。固有周波数測定から逆算することも可能（例えば$f_{01}=2.405c/(2\pi a)$から$c$を求める）。モードの実測スペクトルから張力と密度を求める方程式を立て、最小二乗法で推定する例もある。Bessel零点$\alpha_{mn}$は文献値利用。  

- **実装メモ**  
  小型ドラムでは高速FSH（ファーストサンプル＆ホールド）やFDMによる音響支配波動方程式解法も可能だが、リアルタイム合成ではモード合成が主流である。特に複数の円膜モードと一部の固有共鳴（スネアワイヤ、殻）を並列に合成する構造がリアルタイム向きで、容易に周波数調整・減衰調整ができる。単位モードあたりの計算コストは低いので、90年代以降のドラム物理モデル（FDN,モードサンプリング）が実用例に多い。ドラムヘッドのFDMやFDTDは計算量が高いため、本格的物理シミュレーションを行う場合は非実時間で行われることが多い。
